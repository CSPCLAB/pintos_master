## 2. **세부 설명**

실제 리눅스 시스템의 부팅과정은 다음과 같다.

1. BIOS
2. Boot Loader
3. Kernel
4. Upstart

해당 부팅 프로세스가 완료되면 파일시스템 접근, 유저 프로그램 실행이 가능한 idle 상태로 전환된다.

### BIOS

컴퓨터에 전원이 공급되면 메인보드의 ROM에서 실행되는 firmware이다. 부팅에서 BIOS의 역할은 disk상에 MBR에 있는 boot loader를 호출하여 실행하는 것이다.
